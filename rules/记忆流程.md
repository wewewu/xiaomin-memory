# 记忆流程规则

## 每日记忆流程

### 1. 对话中自动记录
- 重要信息直接记录到对应分类
- 使用 `clawvault capture` 或直接创建 .md 文件

### 2. 手动分类
将记忆分类到：
- `people/` - 人物信息
- `lessons/` - 教训和经验
- `preferences/` - 偏好设置
- `decisions/` - 重要决策
- `projects/` - 项目追踪
- `patterns/` - 模式规律
- `goals/` - 目标

### 3. 定期整理
- 每天检查 inbox/
- 每周分类整理

## 常用命令

```bash
# 快速记录
clawvault capture "今天学到了..."

# 搜索
qmd search "关键词"

# 查看统计
clawvault stats

# 更新索引
qmd update
```
